# jar

"jar" or Just-Another-Renderer is an Image-to-Image pipeline aimed at generating and rendering 3D models of segmented parts of 2D images. 

### Expected Libraries to be Used:
1. SAM2 (Segment Anything Model) - Meta
2. InstantNGP
3. Point3D
4. PyTorch
5. PyOpenGL
6. PyQt
